# Next.js App Router Resume

## Description
A single-page application built with Next.js (App Router) and styled exclusively with Tailwind CSS. This project serves as a digital resume showcasing my education, projects, and technical skills.

## How to run locally
1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000)

## Features Implemented
- [x] Scaffolded Next.js App Router project
- [x] Recreated resume using semantic HTML elements
- [x] Styled layout using Tailwind CSS utility classes (no custom CSS)
- [x] Page loads without browser console errors
